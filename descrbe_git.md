# Начальное знакомство с командами GIT

1. **git init**

2. **git add (имя файла)**

3. **git commit -m "комментарий"**

4. **git diff**

5. **git log**

6. **git checkout (название ветки и комита)**

## Работа с ветками

Чтобы вывести на экран список коммитов в виде дерева, оспльзуйте команду __git log --graph__
